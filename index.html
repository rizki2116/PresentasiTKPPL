React

Apa Itu React?

React adalah library JavaScript bersifat open-source yang digunakan untuk membangun antarmuka pengguna (user interface).

React bekerja berdasarkan component-based. Dimana dalam membuat web aplikasi pada react terdiri dari komponen-komponen yang nantinya digabung menjadi satu.

Apa Itu React?

React bersifat one way data flow, yaitu data atau step yang disimpan lebih terarah sehingga saat proses pembuatan interface jauh lebih mudah.

Learn Once, Write Anywhere

Developer dapat men-develop fitur baru menggunakan react tanpa mengubah kode sebelumnya, react juga dapat bekerja menggunakan Node JS dan mobile apps menggunakan React Native.

Kelebihan React

1. Mudah dipelajari dan Dipahami

ReactJS mudah dipelajari karena memiliki penulisan code yang deklaratif. Akan semakin mudah memahami ReactJS jika sudah menguasai dasar-dasar Javascript karena konsep yang digunakan mirip.

2. Dapat dipakai berulang

ReactJS memungkinkan kita untuk menggunakan lagi beberapa kode atau semua yang telah dikembangkan ke aplikasi lain sehingga pekerjaan lebih cepat selesai.

3. Fitur JSX

ReactJS menggunakan JSX dimana menulis komponen menjadi lebih sederhana karena memungkinkan untuk menggabung HTML dengan javascript.

4.DOM virtual

Dengan hadirnya DOM Virtual membuat kita dapat membangun DOM virtual dan menghosting ke dalam memori sehingga website terlihat reaktif.

5. SEO Friendly

ReactJS mendukung SEO, ketika menggunakan library javascript lain, hasilnya akan berbeda dengan saat menggunakan ReactJS, karena javascript yang Anda pakai dari library ini umumnya dapat langsung diterima browser sebagai halaman situs.

Kekurangan React

1. View Layer

   React JS merupakan library javascript dalam bentuk view layer. Framework ini tidak cocok untuk membangun aplikasi dalam skala besar. Kita perlu melengkapinya dengan router, data layer, struktur dan lainnya secara terpisah.

2. Dokumentasi yang Berantakan

   Dokumentasi pada ReactJS dikatakan kurang bagus karena informasi yang diberikan tidak lengkap. Walau demikian, framework satu ini masih dapat dengan mudah digunakan jika kita rajin mempelajarinya.

3. Dukungan Browser

   Framework ini hanya bisa digunakan pada browser versi terbaru dan tidak bisa digunakan pada versi lawas. Maka, sebelum menggunakan framework ini kita perlu memerhatikan browser yang kita gunakan.

Performansi

Keuntungan besar menggunakan React adalah kinerjanya. React ini dengan cepat menjadi terkenal karena kecepatannya yang luar biasa dan banyak fitur yang memungkinkannya, seperti:

  • Menggunakan DOM virtual untuk efisiensi maksimum dengan hanya merender ulang node sesuai kebutuhan
  • Dukungan untuk bundling dan tree-shaking untuk meminimalkan beban sumber daya pengguna akhir
  • Dukungan untuk rendering sisi server (SSR) untuk kinerja, visibilitas mesin pencari, dan berbagi sosial

Dengan menggunakan DOM virtual, pengembang dapat dengan mudah meningkatkan kinerja dan mengoptimalkan beban kerja aplikasi. Tidak seperti Angular, pengikatan data satu arah React memastikan aliran data dalam satu arah saat menulis aplikasi React, memberikan kontrol yang lebih baik kepada pengembang atas proyek penuh.

React juga mudah diuji. Saat menguji komponen, React memonitor output dari tindakan yang dibuat, membuatnya lebih mudah diakses sehingga Anda dapat memiliki kontrol lebih besar atas bagaimana kode dieksekusi.

Dalam istilah yang lebih sederhana, DOM dalam React adalah standar yang mendefinisikan bagaimana Anda mendapatkan, memperbarui, atau memanipulasi elemen HTML. React DOM tahu cara membuat halaman tetapi tidak begitu cerdas untuk melacak setiap node dan komponen yang diperbarui.

Ketika pengguna berinteraksi dengan UI, React akan membuat salinan dalam bentuk Virtual DOM dan memperbarui UI setelah membandingkan DOM nyata dengan DOM virtual. Dengan cara ini, item yang perlu diperbarui akan diperbarui tanpa mempengaruhi item lainnya.
