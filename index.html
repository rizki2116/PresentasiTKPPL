React

Apa Itu React?

React adalah library JavaScript bersifat open-source yang digunakan untuk membangun antarmuka pengguna (user interface).

React bekerja berdasarkan component-based. Dimana dalam membuat web aplikasi pada react terdiri dari komponen-komponen yang nantinya digabung menjadi satu.

Apa Itu React?

React bersifat one way data flow, yaitu data atau step yang disimpan lebih terarah sehingga saat proses pembuatan interface jauh lebih mudah.

Learn Once, Write Anywhere

Developer dapat men-develop fitur baru menggunakan react tanpa mengubah kode sebelumnya, react juga dapat bekerja menggunakan Node JS dan mobile apps menggunakan React Native.
